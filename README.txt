
Projektarbeit Portfolio - StackBlitz template
--------------------------------------------
Files included:
- package.json
- index.html
- src/main.jsx
- src/App.jsx
- src/styles.css
- public/photos/photo1.png ... photo4.png

How to use on StackBlitz:
1. Go to https://stackblitz.com/
2. Click Create Project -> JavaScript -> Vite + React
3. In the file explorer, replace the project files with the contents of this template:
   - Copy package.json, index.html, src/, public/
4. Open Terminal in StackBlitz and run: npm install
5. Then run: npm run dev

Note: Replace the EmailJS placeholders in src/App.jsx with your EmailJS SERVICE_ID, TEMPLATE_ID, and PUBLIC_KEY.
